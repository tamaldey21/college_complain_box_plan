Admin Login Help

To log in as an administrator, you need:

  • A valid college email address
  • Your assigned password
  • Your specific role in the college administration

If you're having trouble logging in:

  • Ensure you're using the correct email and password
  • Select the appropriate role from the dropdown
  • Contact the system administrator if you continue to have issues
